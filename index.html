# 스터디 진행 방식

스터디 진행은 GitHub를 통해서 이루어집니다.

  • Algorithm Repository : 알고리즘 풀이한 내용을 관리합니다.

    • Pull Request 에 대한 사용방법을 숙지하고 있어야합니다.
    • 프로그래머스, Codlity 등의 사이트에서 공부합니다.

  • TodayReview(일일회고) : 간단하게 일일회고를 작성합니다.

    • 특이사항이 없는 경우 Issue Tab 을 이용합니다.

  • JavaScript Repository : Javascript와 관련된 내용을 기록합니다.

  • InfoBoard : 개발과 관련된 각종 정보에 대해 기록합니다.
